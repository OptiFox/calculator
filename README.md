# The Odin Project: Calculator

## Project Overview

This repository contains my solution for the calculator project by [The Odin Project](https://www.theodinproject.com/lessons/foundations-calculator), as the final project of **Foundations**.

## Features

This page contains the following:

- **A fully functional calculator:** Every button is interactive, and you can see the output from the display.
- **Keyboard support:** You can use numbers, and all the operator keys included in this calculator from the keyboard.
- **Proper error handling:** If you divide anything with 0, you will get an error message.

## Technologies used:


* **HTML5:** For the buttons and main page layouts.
* **CSS3:** For the calculator to look aesthetically pleasing, and accurate.
* **JavaScript:** For all the functionality and interactive web experience.
